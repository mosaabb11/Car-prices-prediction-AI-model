# Car Price Prediction using Machine Learning

## Overview
This project predicts the selling price of used cars using the CarDekho dataset. It demonstrates the complete machine learning workflow, including data cleaning, preprocessing, feature engineering, model training, and evaluation.

## Dataset
- Source: CarDekho Vehicle Dataset
- Target Variable: `selling_price`

## Preprocessing
- Removed duplicate records
- Handled missing values
- Cleaned numerical columns (mileage, engine, max_power, torque)
- Encoded categorical features
- Applied StandardScaler
- Split data into training and testing sets

## Models Used
- Linear Regression
- Support Vector Regression (SVR)
- Decision Tree Regressor
- Random Forest Regressor
- AdaBoost Regressor

## Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
